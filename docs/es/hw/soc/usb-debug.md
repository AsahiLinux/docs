---
title: Interfaz de Depuración USB-PD
---

Parece ser una interfaz de depuración USB integrada.

Comandos a través de EP 0x01, respuestas a través de 0x81, 0x82 desconocido

Notas de ingeniería inversa intentando enviarle datos aleatorios:

```
Los primeros 3 bytes se ignoran. El cuarto byte importa.
Los primeros 5 bytes se repiten al inicio de las respuestas
Las respuestas parecen seguir un formato uniforme

> 00000000 000c8000 00000000 00000000
             OOLLLL

Lee LLLL bytes (redondeados a dword) comenzando desde el registro dword OO.

Retorna:
00000000 00000000 <u32:longitud de datos> <palabras de datos>* <u32:tamaño de datos> <u32:estado=4?>

> 00000000 02000000 00000000 00000000 00000000 00

Retorna:
    00000000 02000000 08458400

Y luego inunda [a0030000] repetidamente para siempre

Parece ser una longitud insana. ¿Error?
```
